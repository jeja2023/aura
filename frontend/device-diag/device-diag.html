海康设备接入与抓拍推送说明

      

      

        
本页聚焦运维 ISAPI 联调（浏览器经后端访问 NVR）。如需业务抓拍入库说明，请前往导航「NVR 设备」页（/device/）。

        
    
          
  1. 在平台登记 NVR：超级管理员在「NVR 设备管理」页使用「注册新设备」填写名称、IP、端口、品牌与协议；成功后得到 deviceId。
    2. 
          
  2. 在库中为该设备配置 hmac_secret（生产环境必须配置；开发环境可按后端策略回退）。抓拍接口将校验 X-Signature。
    3. 
          
  3. 将海康或网关的 HTTP 推送地址指向：
    4. 
          
  4. 请求：POST，Content-Type: application/json。请求体字段与适配器一致：deviceId（数字）、channelNo（与布点一致）、captureTime（ISO 8601，建议带时区）、imageBase64（纯 Base64，无 data: 前缀）、其它元数据可一并放入 JSON（勿重复塞大图）。
    5. 
          
  5. 请求头 X-Signature：对字符串 deviceId|channelNo|captureTime（其中 captureTime 与 JSON 内值字节一致，采用 .NET 默认 O 格式）做 HMAC-SHA256，密钥为设备 hmac_secret，输出小写十六进制字符串。
    6. 
          
  6. 若启用抓拍 IP 白名单，仅允许名单内地址调用 /api/capture/push。
    7. 
          
  7. 通道号与 ISAPI「抓图通道序号」对应关系请用下方「模拟通道列表 / 代理通道」对照 NVR 实际配置，再与「摄像头布点」中的 channelNo 对齐。
    8.
