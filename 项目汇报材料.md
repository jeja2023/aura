# 寓瞳 (Aura) 智能集宿区人员行为研判平台项目汇报

## 一、 项目定位
**寓瞳 (Aura)** 是一款聚焦于**集宿区、公寓及封闭式园区**安全防范与人员考勤管理场景，端到端的**被动无感知智能视觉解析与人员追踪平台**。
系统基于轻量级三维数字孪生与多维空间碰撞引擎，集成多协议设备接入、Re-ID 特征提取与聚类、高维向量检索等核心能力，实现对人员的时空动线还原与复杂行为的主动研判。

## 二、 解决的核心问题
1. **打破异构设备壁垒**：传统安防平台与特定厂商硬件深度绑定。本项目基于标准适配器架构，支持海康威视、大华、宇视等多品牌 NVR 和通用 ONVIF 协议设备统一低损接入。
2. **弥补集宿区场景研判空白**：针对“晚归/夜不归宿考勤”、“群租房检测”、“外来异常滞留”等细分管理痛点，填补了通用安防平台缺乏垂直业务逻辑的缺失。
3. **实现被动无感知管理**：改变了依赖人员主动配合（刷卡、站定刷脸等）的传统考勤方式，基于自然过镜抓拍进行被动识别，有效防范尾随、代刷卡等管理漏洞。
4. **显著降低建设成本**：采用全开源核心技术栈，无需购买高昂的商业 AI 中台授权与专用智能硬件，支持在局域网纯内网物理隔离部署，保障核心图片数据安全不出网。
5. **规避复杂运维瓶颈**：支持多 AI 推理节点无 Nginx 直连轮询、熔断与限流保护，支持管理员在前端进行推理节点的在线热更新，极大降低了现场的部署和运维成本。

## 三、 核心落地功能
1. **多协议接入与告警长链**：支持 ISAPI Webhook、C++ SDK 及 ONVIF 多通道接入，内置海康告警长连接监听服务，实现设备状态诊断与主动告警推送。
2. **AI 特征链路与无监督聚类**：自动提取人脸及人体 512 维深度特征向量，结合 ArangoDB 向量数据库实现百万级实时以图搜人，基于无监督聚类算法生成虚拟身份 ID 档案。
3. **空间引擎与 ROI 防区编辑**：支持上传楼层图纸，Canvas 拖拽布点与自定义多边形 ROI 防区绘制，建立“摄像头-ROI防区-物理房间”的空间拓扑关联。
4. **业务智能研判引擎**：内置落脚宿舍研判、夜不归宿考勤异常告警、多 ID 停留群租告警等定时分析任务。
5. **3D/2D 混合态势大屏**：使用 Three.js 渲染轻量 3D 楼宇白模，结合 SignalR 实时推送。支持 3D 告警跨层联动与 2D 平面切片无缝下钻，提供 Canvas 平面轨迹带箭头动线播放。
6. **数据外联与审计底座**：提供操作与系统双重审计日志，并通过 JSON 单向输出通道向内网大数据平台低损耗供数，支持标准的 RBAC 组织权限管控。

## 四、 未来计划与优化方向
1. **长链事件与流媒体管道拆分**：将常驻的设备事件订阅（如 ISAPI 长连接长读服务）与视频取流回放媒体通道进行微服务化独立拆分，降低主 WebAPI 服务的超时与资源争抢风险。
2. **AI 聚类算法演进与防漂移**：引入“人工辅助合并”与“定期重聚类”机制，降低聚类漂移对轨迹和研判准确率的影响；优化多摄像头交叉验证算法，在 ROI 遮挡场景提高研判置信度。
3. **大规模多节点编排与可靠性**：优化推理集群的容器化部署（如 Kubernetes 编排与 HPA 自动扩缩容），提升在大规模节点并发推送下的 Redis 消息队列防积压和断网重发机制。
4. **自动化监控与安全加固**：集成 Prometheus 性能指标抓取与 OpenTelemetry 链路追踪；全量推行 HttpOnly Cookie 会话，并加强上线 readiness 就绪自检在 CI 流程中的自动阻断机制。
