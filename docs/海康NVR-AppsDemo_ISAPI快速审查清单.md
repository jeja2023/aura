## 四、与当前项目字段映射模板

| 海康侧字段 | Aura目标字段 | 说明 |
|---|---|---|
| 设备IP/序列号 | `deviceId` | 通过平台设备表做映射，不直接入库IP作为主键 |
| 通道号（如101） | `channelNo` | 需先确认编号规则并固化映射 |
| 事件时间 | `captureTime` | 建议统一 ISO8601，保留时区 |
| 图片二进制/JPEG | `imageBase64` | 入站前转 Base64 |
| 事件扩展字段 | `metadataJson` | 保留原始关键信息，避免丢字段 |
